# Simple To Do Planner

A clean and modern todo list application built with HTML, CSS, and JavaScript.

## Features

-  **Add & Manage Tasks** - Easily add, complete, and delete tasks
-  **Persistent Storage** - Tasks are saved locally using localStorage
-  **Mobile Responsive** - Optimized for all screen sizes
-  **Performance Optimized** - Uses event delegation to prevent memory leaks
-  **Clear Completed** - Bulk remove all completed tasks with one click

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Bootstrap**
- **Bootstrap Icons**
- **LocalStorage API**

Built as part of a Full Stack Development Course
